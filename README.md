# Premium Portfolio - Miguel Ángel Ramírez Corredor

¡Bienvenido a mi portafolio profesional! Este proyecto es una vitrina digital diseñada con una estética moderna, minimalista y de alto nivel, enfocada en resaltar mi trayectoria como **Técnico en Programación de Software** y **Estudiante de Ingeniería Informática**.

## 🚀 Vista Previa
El sitio cuenta con un diseño de una sola página (one-page) que incluye:
- **Hero Section**: Introducción impactante con fotografía profesional.
- **Sobre Mí**: Resumen de trayectoria e intereses profesionales (IA, Salud Mental, Full Stack).
- **Estudios**: Formación técnica y académica.
- **Proyectos**: Sección de trabajos destacados con detalles expandibles (Modal de Mogi IA).
- **Habilidades**: Stack tecnológico moderno.
- **Contacto**: Enlaces directos a Correo y GitHub.

## 🛠️ Tecnologías Utilizadas
Para el desarrollo de este portafolio se emplearon herramientas de vanguardia:
- **HTML5 & CSS3**: Estructura semántica y diseño personalizado.
- **Bootstrap 5**: Framework para un sistema de grid robusto y componentes responsivos.
- **Glassmorphism & Dark UI**: Estética moderna utilizando `backdrop-filter`, gradientes de malla y efectos de brillo.
- **AOS (Animate On Scroll)**: Animaciones fluidas al navegar por el sitio.
- **JavaScript (Vanilla)**: Lógica para efectos de scroll, navegación y gestión de modales.

## 📁 Estructura del Proyecto
El código está organizado de manera limpia y modular:
```
parcial2web/
├── index.html          # Estructura principal del sitio
├── css/
│   └── style.css       # Estilos personalizados (Glassmorphism, Colores, Tipografía)
├── js/
│   └── script.js       # Interactividad y efectos
├── img/
│   ├── profile.jpg     # Fotografía de perfil
│   └── mogi_ia.png     # Imagen del proyecto Mogi IA
└── README.md           # Documentación del proyecto
```

## ⚙️ Cómo Ejecutar
1. Clona el repositorio o descarga los archivos.
2. Asegúrate de mantener la estructura de carpetas (`css/`, `js/`, `img/`).
3. Abre el archivo `index.html` en cualquier navegador web moderno (Chrome, Edge, Firefox).
